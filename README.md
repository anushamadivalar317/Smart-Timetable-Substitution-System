# Smart Scheduling, Timetable Generation, and Faculty Substitution Management System

## Overview

A Django-based web application for managing teacher schedules, attendance, and timetable substitutions.

## Features

* Teacher Registration
* Teacher Login
* Attendance Tracking
* Admin Dashboard
* Teacher Dashboard
* Certificate Management
* Timetable Substitution Management

## Technologies Used

* Python
* Django
* SQLite
* HTML
* CSS
* Bootstrap

## Project Structure

* members/
* certifications/
* static/
* test1/
* manage.py

## How to Run

1. Create and activate a virtual environment
2. Install Django
3. Run migrations

python manage.py migrate

4. Start the server

python manage.py runserver

5. Open:
http://127.0.0.1:8000


http://127.0.0.1:8000
